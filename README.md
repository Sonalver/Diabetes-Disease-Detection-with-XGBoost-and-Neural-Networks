# Diabetes-Disease-Detection-with-XGBoost-and-Neural-Networks-Logistic-Regression
Diabetes mellitus, commonly known as diabetes, is a group of metabolic disorders characterized by elevated blood sugar levels. Diabetes Disease Detection with XGBoost and Neural Networks.
This project includes:

✔ Data analysis
✔ Data visualization
✔ Feature scaling
✔ Train–test data split
✔ ANN deep learning model
✔ XGBoost classifier
✔ Logistic Regression classifier
✔ Confusion matrix & classification report
✔ Accuracy comparisons

Containing 8 input features:
Pregnancies
Glucose
BloodPressure
SkinThickness
Insulin
BMI
DiabetesPedigreeFunction
Age

Target variable:
Outcome (0 = No diabetes, 1 = Diabetes)

Models Implemented
1️. Artificial Neural Network (ANN)
2 hidden layers (400 neurons each)
ReLU activation
Dropout (0.2)
Sigmoid output layer

2️.XGBoost Classifier
max_depth = 5
n_estimators = 100
learning_rate = 0.1

3️.Logistic Regression
Baseline ML model for comparison

Evaluation Metrics
Each model is evaluated using:
Accuracy Score
Confusion Matrix
Classification Report
Precision
Recall
F1 Score

Outputs & Visualizations
Outcome distribution plot
Feature pairplot
ANN training loss curve
ANN training accuracy curve
Confusion matrices
Model classification reports
